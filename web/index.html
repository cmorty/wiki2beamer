wiki2beamer

collaborative slide editing made easy

wiki2beamer converts a simple wiki-like syntax to complex LaTeX beamer code. It's written in python and should run on windows and all *nix platforms.

Download

Get the latest version of wiki2beamer at sourceforge.net.
